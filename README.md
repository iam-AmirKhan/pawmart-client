## About PawMart

PawMart is your one-stop online pet store offering a wide range of premium pet products, accessories, and essentials for your beloved furry friends. Built with modern web technologies to provide a seamless shopping experience for pet lovers.

## Key Features

- **Extensive Product Catalog** - Browse through hundreds of quality pet products including food, toys, accessories, and healthcare items for dogs, cats, and other pets

- **Smart Search & Filtering** - Advanced search functionality with category filters, price ranges, and sorting options to help you find exactly what your pet needs

- **Dark/Light Theme Toggle** - Comfortable browsing experience with a beautiful theme switcher that adapts to your preference and saves your choice

- **Responsive Design** - Fully optimized for all devices - desktop, tablet, and mobile - ensuring smooth shopping from anywhere

- **User-Friendly Interface** - Intuitive navigation with clean design built using React and Tailwind CSS for fast and engaging user experience

## Technologies Used

- **Frontend:** React.js, Tailwind CSS, DaisyUI
- **Icons:** Lucide React
- **Routing:** React Router
- **State Management:** React Hooks
- **Deployment:** Netlify
